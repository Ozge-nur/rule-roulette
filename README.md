# 🌟 Yıldız Avcısı

Yıldızları topladıkça Yapay Zeka'nın size anlık yasaklar koyduğu, sınırları zorlayan tarayıcı tabanlı bir refleks/bulmaca oyunudur. Hiçbir kurulum gerektirmez.

Nasıl Oynanır?
*   Amaç: Haritadaki Yıldızları (🌟) toplayarak skoru artırmak. (WASD veya Ok Tuşlarını kullanın).
*   Yapay Zeka Kuralları:Her yıldız aldığınızda ekrana süreli bir yasak gelir (Örn: Sola gitmek yasak). Yasağı delerseniz **2 saniye donma cezası** yersiniz.
*   Bombalar:Skid arttıkça harita mayın (💣) tarlasına döner. Bombalara basarsanız ölürsünüz.
*   İpucu: Ekranın sağından çıkarsanız solundan girersiniz! Kaçmak için bunu kullanın.*
